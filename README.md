# React Routing Dashboard Assignment

A responsive dashboard application built with React, React Router, Ant Design, and Tailwind CSS. The dashboard provides navigation between multiple mini projects including:

- 🔍 GitHub User Finder
- 🌦️ Weather App
- 📝 Todo App
- 🛍️ Noon Website UI

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Ant Design
- React Icons

## Features

- Responsive Dashboard Layout
- Sidebar Navigation
- Dynamic Routing
- Mobile-Friendly Design
- Modern UI
